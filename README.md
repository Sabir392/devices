## Usage

1. Run `EPM-310-RS-485.exe`
2. Browser opens automatically
3. Select COM port and Baud Rate in settings
4. Click **Connect**

## Requirements

- Windows 7 / 10 / 11 (64-bit)
- USB-RS485 adapter (CH340)
- EPM-310 device
